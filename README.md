## electron-vue-project

> 自己学习electron-vue 的测试例子
>
> 一个基于electron-vue开发的音乐播放器部分功能、内嵌网易云nodejs版api服务调用网易云api。

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
